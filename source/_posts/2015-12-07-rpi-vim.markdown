默认树莓派没有安装vim，执行下面命令安装之

	sudo apt-get install -y vim

安装好之后就可以使用vi或者vim进行编辑操作，但是没有代码高亮显示。

<!--more-->

在`~`目录下面新建.vimrc文件可以实现
```
pi@raspberrypi ~ $ cd ~
pi@raspberrypi ~ $ vim .vimrc
```

在.vimrc文件中插入以下代码：

```
set nu
syntax on
set tabstop=4
```
再试试vim，看看是不是就有代码高亮了呢~
