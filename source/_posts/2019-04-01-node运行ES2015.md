# 添加.babelrc

```json
{
  "presets": [
    "es2015"
  ]
}
```

# 安装插件

* yarn

```bash
yarn add babel-preset-es2015
```

* npm

```bash
npm install babel-preset-es2015 --save
```

# 以babel-node运行

运行命令后边添加`--exec babel-node`
